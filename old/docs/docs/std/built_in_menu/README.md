>## 操作说明  
键盘开机后 **长按旋钮** 进入内置菜单  
进入菜单后 **单击旋钮确认** ，**旋转旋钮滚动选择**  

**阅读本文档时若发现任何与你的菜单不符的情况请 [检查固件更新](/docs/std/web_hid/README.md)**  
**若已是最新版本请联系小夜催更本文档**
