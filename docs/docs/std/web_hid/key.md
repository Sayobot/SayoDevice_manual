
# 按键绑定

> # 设置按键  

1. 按键模式选择 **默认**，  
2. 点击下面的框选择键盘按键  
- **Ctrl Shift Alt Win** 可以 **不选、选择一个或多个**。  
- **其他按键** 可以 **不选或选择一个**  
- 再次点击按键可以取消选中  

- **设置字母 A**  
![](/img/A.jpg)

- **设置 Ctrl 键**  
![](/img/Ctrl.jpg)  

<details>
<summary>触发原理</summary>  

  *触发普通键后至少保持20ms，直到按键松开*  
  *如果没有延时，某些软件可能无法正常识别按键。*  
  *如果你需要更短的保持时间，请使用**键盘**模式，它没有额外的延时。*  
  *如果你不理解，请忽略这个提示。*  

</details>

> # 设置组合快捷键

- **设置 复制 快捷键 Ctrl+C**  
![](/img/Ctrl+C.jpg)

- **设置 截图 快捷键 Win+Shift+S**  
![](/img/win+shift+s.jpg)

<details>
<summary>触发原理</summary>  

  *在此模式，设置快捷键将会在修饰键后插入20ms延时，然后再触发普通键。*  
  *触发普通键后，至少保持20ms，直到按键松开*  
  *如果没有延时，某些软件可能无法正常识别快捷键。*  
  *如果必须同时触发全部按键，请使用**键盘**模式，它没有额外的延时。*  
  *如果你不理解，请忽略这个提示。*  

</details>

> # 设置两组连续的按键操作  

!> 此模式按下后执行一组快捷键，松开按键后执行另一组快捷键  

1. 按键模式选择 **键盘二步操作**，  
2. 设置两组按键即可，方法同上  

- **设置一键粘贴+回车**  
![](/img/2setp_paste+enter.jpg)  

<details>
<summary>触发原理</summary>  

  *流程：修饰键->20ms->普通键->30ms->等待松开按键->普通键->30ms*

</details>

> # 短按长按功能  

!> 此模式不适合旋钮

1. 按键模式选择 **长/短按**，  
2. 然后设置两组按键即可，方法同上  

- **设置短按A长按B**  
![](/img/tap_and_hold.jpg)  

- **设置短按暂停长按下一曲**  
![](/img/tap_and_hold_mu.jpg)  

<details>
<summary>触发原理</summary>  

  *1.当按键按下时开始倒计时300ms，直到超时或者按键松开*  
  *2.超时触发长按，否则触发短按*

</details>

> # 鼠标模式  
此模式可以设置一个按键为**鼠标左键 鼠标右键 滚轮按下** 或 **光标上下左右移动 滚轮上下滚动**  

- **设置鼠标左键**  
在 **鼠标按键** 选项框中选择按键。下面三个数值设为0  
![](/img/mouse_key.jpg)  

- **设置光标下移**  
在 **鼠标按键** 选项框中取消全部按键，然后把 **鼠标移动 Y** 输入框填 -1  
![](/img/mouse_move_down.jpg)  

- **设置滚轮下滚**  
在 **鼠标按键** 选项框中取消全部按键，然后把 **滚动** 输入框填 -1  

```
横向 正数右移，负数左移  
纵向 正数上移，负数下移  
滚轮 正数前滚动，负数后滚动（MAC相反）  
数值越大移动速度越快。  
当按住按键时，间隔50ms连续移动  
```

> # 键盘+LED控制  

!> *此模式在触发快捷键的前后，还会触发指示灯相关的按键来更新相应的状态*  
*快捷键可以不选，达到只切换键盘灯光状态的效果*  
*包括 Num lock、CapsLock、Scroll Lock*   
- **打开numlock指示灯并关闭CapsLock指示灯**  
![](/img/key_and_led.jpg)  

<details>
<summary>触发原理</summary>  

  *1.打开需要打开的指示灯（自动发送对应按键）*  
  *2.发送设置的按键*  
  *3.关闭需要关闭的指示灯（自动发送对应按键）*  

</details>

> # A+B+C+D
*此模式可以依次按下选中的按键，然后一起松开*  
*适用于需要严格控制按键顺序的快捷键，或者较短的按键集。*  
*最多选择4个按键*  


> # 一键密码
*此模式可以依次输入预设好的ASCII字符串*  

- **设置 一键输入123456后回车**  
1. 在**资源->密码**里面输入内容  
![](/img/pwd_data.png)  
2. 在**按键绑定**里面，把按键模式改成**一键密码**，选择保存的密码编号  
![](/img/pwd.jpg)  
3. 附加按键  
在密码后面可以插入一个任意键，比如回车。  
![](/img/pwd_key.jpg)  
4. 输入间隔  
密码输入速度，默认值   
一般不需要修改。如果发现输入会漏键需要加大这个数值  
蓝牙模式建议改成10以上  
![](/img/pwd_time.jpg)  
5. 锁定大小写  
输入密码的时候自动锁定键盘大写，开启此功能后密码输入才不会被输入法影响。 
![](/img/pwd_case.jpg)  

- 其他键盘布局  
默认键盘布局是ANSI，如果需要改为AZERTY布局，请点击“设备选项”开启

<details>
<summary>触发原理</summary>  

  *1.调整大小写锁定状态（自动发送对应按键）*  
  *2.依次发送设置的字符按键*  
  *3.还原大小写锁定状态（自动发送对应按键）*  

</details>

> # 一键网页

适用于windows系统，设置方法同一键密码  

<details>
<summary>触发原理</summary>  

  *1.按Win+R打开运行，同时开启大小锁定（自动发送对应按键）*  
  *2.延时一段时间，等待运行窗口打开（约500ms）*  
  *3.依次发送设置的字符按键，并回车*  
  *4.还原大小写锁定状态（自动发送对应按键）*  

</details>

> # 一键写诗
一键输出一串文字

!> 部分设备不支持此功能  
**可能存在快捷键冲突。**  
适用于windows系统  

设置方法类似一键密码  但是在**文本**选项卡里面输入内容  
支持两种编码方式（GB18030\UTF16-LE）  
不同软件需要的编码可能**不一样**，如果输入乱码可以**切换另一种编码**  

<details>
<summary>触发原理</summary>  

*按住Alt+小数字键盘直接输入字符编码。*  

</details>

> # 游戏控制器按键

!> 这个属于额外的加载项，仅适用于windows。mac系统 **不要** 开启。  
如果误开启，需要使用windows系统关闭。  

在**设备选项**里的**HID功能**开启 **joystick** 保存重插，  
按键模式改为 **游戏控制器按键**，  
然后输入按键编号（0~127有效）。  
这个按键和游戏手柄的任何按键都没关系，需要游戏支持自定义按键才行。

> # 按键分层（FN）  

!> 部分设备不支持此功能

1. 将一个按键模式改成**FN**并选择需要切换的目标层级  
2. 按下这个按键，键盘会切换到另一个层的功能  
3. 松开这个按键，回到原来的层  

![](/img/FN.jpg)  

<details>
<summary>触发原理</summary>  

  *1.暂存当前层级编号*  
  *2.切换到目标层级编号*  
  *3.等待按键释放*  
  *4.切换到原来的层级*  

</details>
