> ## 设置鼠标光标移动、滚轮

按键模式改为 **鼠标**，  
在 **鼠标键**  不要选择按键，  
**X** 是横向移动，负数左移。正数右移。数字越大速度越快。  
**Y** 是竖向移动，负数上移，正数下移。数字越大速度越快。  
**Scroll** 是鼠标滚轮。负数下滚动，正数上滚动。
